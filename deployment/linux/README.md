# Linux Deployment

## Overview

This directory contains Linux deployment resources for ARGUS.

Linux is one of the primary target environments for future embedded and appliance-based deployments.

---

## Deployment Targets

* Ubuntu
* Debian
* Raspberry Pi OS
* Embedded Linux distributions

---

## Future Appliance Compatibility

ARGUS is intended to operate on dedicated hardware platforms running Linux-based operating systems.

Examples:

* Raspberry Pi 5
* Raspberry Pi Compute Module
* NVIDIA Jetson Series
* Custom ARM Devices

---

## Long-Term Goal

Provide a lightweight, autonomous cybersecurity platform capable of protecting networks directly at the edge.
