# Windows Deployment

## Overview

This directory contains Windows-specific deployment files for the ARGUS prototype.

ARGUS is designed as a platform-independent cybersecurity solution. While the current implementation is primarily validated on macOS, Windows deployment support is actively being developed.

---

## Purpose

The Windows deployment package allows users to:

* Launch the ARGUS detection engine
* Monitor threats in real time
* Access the dashboard interface
* Demonstrate the ARGUS workflow

---

## Future Support

Planned improvements include:

* Automated installation
* Native Windows service integration
* Enhanced packet capture support
* One-click deployment packages

---

## Note

The current repository represents the software prototype used to validate the future ARGUS cybersecurity appliance.
